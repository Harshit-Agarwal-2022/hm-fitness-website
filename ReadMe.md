# 🏋️ HM Fitness Website

A modern, responsive website for **HM Fitness Gym**, a local gym located in **Pratap Nagar, Jaipur, Rajasthan**.

The website is designed to showcase the gym, membership plans, facilities, and provide an easy way for potential members to get in touch through WhatsApp or phone.

---

# 🎯 Project Goals

* Create a premium and modern online presence for HM Fitness.
* Showcase gym facilities and images.
* Display membership plans clearly.
* Enable users to contact the gym directly through WhatsApp.
* Provide easy access to the gym's location and contact information.

---

# 🛠️ Tech Stack

* HTML5
* CSS3
* Vanilla JavaScript
* Responsive Design
* Google Maps Embed
* WhatsApp Integration
